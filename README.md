# 🎓 Smart Study Assistant - Frontend

A modern and responsive React-based frontend for the **Smart Study Assistant**, an AI-powered learning platform that helps students upload notes, generate concise summaries, take AI-generated quizzes, and track their learning progress.

🌐 **Live Demo:** https://ssa-frontend-rho.vercel.app/

---

## 🚀 Features

- 🔐 User Registration & Login
- 📄 Upload PDF and TXT study materials
- 🤖 AI-powered note summarization
- 📝 Automatic quiz generation from uploaded notes
- 📊 Quiz result tracking
- 📱 Fully responsive user interface
- ⚡ Fast and intuitive user experience

---

## 🛠️ Tech Stack

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Study-Assistant-Frontend.git
```

### Navigate to the project

```bash
cd Smart-Study-Assistant-Frontend
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```

---

## 🔗 Backend Repository

This frontend communicates with the Smart Study Assistant backend REST API.

Make sure the backend server is running before using the application.
